About us

The PSI-MS working group defines community data formats and ontology terms facilitating data comparison, exchange and verification. PSI-MS is focused on the field of mass spectrometry.

Our three main achievements and projects are:

  • The mzData standard, which captures peaklist data. mzData's aim is to unite the large number of current formats (pkl's, dta's, mgf's, .....) into one; mzData. mzData has been released in a stable production version. mzData is already supported by some software providers. Other companies, including Bruker Daltonics, and Kratos Analytical (Shimadzu Group), are currently implementing the mzData standard.
  • The analysisXML standard, which captures parameters and results of search engines such as Mascot and Sequest (in addition to many other equivalent tools).
  • The supporting ontology that will provide classifiers for both mzData and analysisXML; this list of clearly defined, contextualised terms is crucial for the construction of unambiguously worded data files.


The mzData standard


Status: stable production version 1.05

mzData in a nutshell

mzData is a data format capturing peak list information. Its aim is to unite the large number of current formats (pkl's, dta's, mgf's, .....) into one; mzData. mzData is NOT a substitute for the rawfile formats of the instrument vendors. Some vendors, if not all, will provide software transforming their raw files to mzData. There are already a number of programs which can use mzData. In order to keep the filesize of mzData limited, mz/intensity information is stored in "binary base 64 format".

The analysisXML standard

Status: conceptual phase
analysisXML captures parameters and results of search engines such as Mascot and Sequest (in addition to many other equivalent tools). analysisXML as a standard will unify results from different search engines, facilitating cross-comparisons. PSI and the major journals are working towards common guidelines for comparing results. Because of the inclusion of search parameters in mzIdent, it can potentially also be used as an input file to a search engine. .
