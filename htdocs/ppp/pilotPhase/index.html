Plasma Proteome Project -

Pilot Phase Data Collection Resources

Introduction

A key aspect of the pilot phase of the Plasma Proteome Project (PPP) is the comparative analysis of the experimental approaches and protein identifications from the participating laboratories. As a prerequisite to the comparative analysis, all data need to be centrally collected. This page describes the resources for the data collection.

Common resources

Shared databases

To be able to compare protein identifications, we need to refer to identified proteins by database accession numbers, not by names or descriptions. Furthermore, the final protein identifications are significantly influenced by the database against which the peptide searches are performed. To improve the comparability of results, we urge all participating labs to perform database searches against the Human section of the International Protein Index (IPI). IPI is a non-redundant data set created from Swiss-Prot/TrEMBL, Ensembl, and RefSeq. It provides a high level of completeness, a minimal level of redundancy, and monthly updates. To have an identical search base, we recommend all labs participating in the PPP pilot phase use IPI human of July 2nd, 2003:  ftp://ftp.ebi.ac.uk/pub/databases/IPI/old/HUMAN/ipi.fasta.v2.21.gz
Please note that this is a human-only dataset, you might need to add specific "contaminants", e.g. trypsin, depending on your experimental protocol.

If the database/search engine you are using is not under your control, please provide Swiss-Prot/TrEMBL accession numbers for the identified proteins.

Data submission

For the data submission, we provide two alternative methods. Participants may either insert their data into Excel forms and submit the forms, or they may submit their data in XML according to a common XML schema. Please choose one of the two methods. XML based submission is the preferred method, because it puts less workload on the central data collection and processing team.

Excel based data submission

We have prepared a set of Excel spreadsheets for data submission. Each sheet is accompanied with a detailed instruction document. Please use only these forms for Excel-based submissions, we won't be able to process forms in a different format. All forms and instructions are available at: http://psidev.sourceforge.net/ppp/pilotPhase/excelForms/
Please note: The forms have been updated with respect to the original forms which have been sent to participants. We prefer submissions in the updated forms.

Please use the submission system at the University of Michigan
http://www.bioinformatics.med.umich.edu/proteome/hupo/ppp/ or send your data to Simone.Pakin@Ludwig.edu.au at the Ludwig Institute.

XML based data submission

To facilitate the processing of submitted data, we encourage data submission in XML. We have developed an XML schema which stores the same data as the Excel forms. The schema is available from http://psidev.sourceforge.net/ppp/pilotPhase/xml/HUPO-PPP-Schema.xsd, the auto-generated schema documentation from http://psidev.sourceforge.net/ppp/pilotPhase/xml/doc/
If bioinformatics support is available in your lab, it is probably the easiest to directly format your data according to the XML form by a script.

For detail editing, or if no bioinformatics support is available, we offer a customised version of the prototype PEDRo tool to provide a user-friendly interface to create the XML formatted data. Please see the README on how to install and use it.

Please send all XML formatted submissions to hhe@ebi.ac.uk at the European Bioinformatics Institute.
